# Placeholder for opendbc repository

The original openpilot project depends on the upstream
[`opendbc`](https://github.com/commaai/opendbc) repository. In this
training environment we cannot fetch the upstream submodule, so the
`opendbc_repo` directory is tracked directly in this repository and
contains a lightweight stub.

To work on features that require the real dataset, replace the contents
of this directory with a checkout of the upstream project.
